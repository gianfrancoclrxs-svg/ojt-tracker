Privacy Policy

OJT Tracker
Effective Date: April 21, 2026

1. Overview

OJT Tracker is a student-developed web application designed to help users track their On-the-Job Training (OJT) progress, including attendance logs, total hours, and company information.

This Privacy Policy explains how user data is collected, stored, and used within the application.

2. Information We Collect

We may collect and store the following data:

Account Information:

  • Full Name
  • Username
  • Email Address
  • Student ID

OJT Information:

  • Company / Location Name
  • Company Address
  • Target OJT Hours

Logbook Data:

  • Attendance logs (date, AM/PM time in/out)
  • Half-day and absent records
  • Total hours and minutes rendered

Feedback Data:

  • Rating (1–5 stars)
  • Written feedback
  • Submission date

3. How We Use Your Information

All collected data is used only for app functionality, including:

  • User authentication and account management
  • Tracking OJT attendance and progress
  • Displaying logbook records and total hours
  • Storing user feedback for app improvement

We do not use your data for advertising, profiling, or marketing purposes.

4. Data Storage & Security

Your data is securely stored using Google Firebase (Firestore), a cloud database service.

While standard security measures are implemented, no system is 100% secure. Users are responsible for keeping their login credentials safe.

5. Data Sharing

We do not sell, rent, or share user data with third parties.

Data is only accessible:

  • To the user (through their account)
  • To the Firebase backend required to operate the application

6. Account Deletion

Users may delete their account anytime through the app settings.

When an account is deleted:

  • All user information is permanently removed
  • All logbook records are deleted
  • Feedback associated with the user is removed (if applicable)

This action cannot be undone.

7. Feedback System

Feedback submitted through the app is stored for improvement purposes and is not publicly displayed outside the system unless used for internal review.

8. Cookies & Local Storage

OJT Tracker uses local storage only for session purposes:

  • Login session management
  • User session ID storage

No tracking cookies or third-party analytics are used.

9. Third-Party Services

This application uses:

  • Google Firebase (Firestore) for authentication and database storage

These services may process data according to their own privacy policies.

10. Disclaimer

OJT Tracker is a student project developed for educational purposes only and is not affiliated with any institution, company, or organization.

The developer is not responsible for data loss, misuse, or system errors.

11. Changes to This Policy

This Privacy Policy may be updated from time to time. Changes will be reflected within the application.
